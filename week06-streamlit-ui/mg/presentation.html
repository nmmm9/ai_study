부족하면 쿼리를 바꿔서 다시 검색 (최대 3회 반복)

Pipeline RAG

검색 1번 → 답변. 결과가 별로여도 그냥 답함.

Agentic RAG

AI가 검색 결과를 평가하고, 부족하면 다른 쿼리로 재검색. 충분해질 때까지 반복.

Thinking 과정

Agent의 판단 과정이 실시간으로 표시됨

✓ 계획 (1/3) — 검색된 청크 0개, 행동 결정 중...
✓ 검색 1 — "스타트업 투자 유치 단계" 검색 중...
✓ 검색 결과 — 새로운 청크 5개 추가 (총 5개)
✓ 평가 — partial (부족: 사업계획서 작성법)
✓ 검색 2 — "사업계획서 구성요소" 검색 중...
✓ 검색 결과 — 새로운 청크 3개 추가 (총 8개)
✓ 평가 — sufficient
● 답변 생성 중...

Planner(행동 결정) → Searcher(검색 실행) → Grader(결과 평가) → 반복 또는 답변
