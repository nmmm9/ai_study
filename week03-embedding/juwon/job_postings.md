## 공고 1: 카카오 - 백엔드 개발자 (신입/경력)

**회사명**: 카카오
**직무**: 서버/백엔드 개발자
**고용형태**: 정규직
**근무지**: 경기도 성남시 분당구 (판교)
**등록일**: 2025-01-15
**마감일**: 2025-02-28

### 직무 소개

카카오 서버 플랫폼팀에서 수억 명의 사용자가 사용하는 카카오톡, 카카오맵, 카카오페이 등의 핵심 백엔드 시스템을 개발하고 운영합니다. 대규모 트래픽을 안정적으로 처리하고, 새로운 기능을 빠르게 제공하는 것이 우리 팀의 목표입니다.

### 주요 업무

- 카카오 서비스의 서버 사이드 API 설계 및 개발
- 대용량 데이터 처리 파이프라인 구축 및 최적화
- MSA(마이크로서비스 아키텍처) 기반의 서비스 개발 및 운영
- 시스템 성능 모니터링 및 장애 대응
- 코드 리뷰 및 기술 문서 작성

### 자격 요건

- Java 또는 Kotlin 기반 백엔드 개발 경험 1년 이상
- Spring Boot, Spring MVC 프레임워크 사용 경험
- MySQL, Redis 등 데이터베이스 사용 경험
- RESTful API 설계 및 개발 경험
- 기본적인 리눅스 운영 환경 이해

### 우대 사항

- 대규모 트래픽 처리 경험 (DAU 100만 이상)
- Kafka, RabbitMQ 등 메시지 큐 사용 경험
- Docker, Kubernetes 기반 컨테이너 환경 경험
- 오픈소스 기여 경험
- 컴퓨터공학 관련 전공자

### 복리후생

- 유연근무제 (코어타임 10:00~16:00)
- 재택근무 가능 (주 2회)
- 식대 지원 (월 20만원)
- 자기계발비 지원 (연 200만원)
- 건강검진 지원
- 카카오 임직원 할인 혜택

### 전형 절차

서류 전형 → 코딩 테스트 → 1차 기술 면접 → 2차 임원 면접 → 최종 합격

**회사명**: 네이버
**직무**: AI/ML 엔지니어
**고용형태**: 정규직
**근무지**: 경기도 성남시 분당구 (그린팩토리)
**등록일**: 2025-01-20
**마감일**: 2025-03-15

- ML 파이프라인 설계 및 MLOps 환경 구축
- 연구 논문 리뷰 및 사내 기술 공유

### 우대 사항

- HuggingFace Transformers 사용 경험
- MLflow, Weights & Biases 등 실험 관리 도구 경험
- 클라우드 환경(AWS, GCP, NCP) 에서의 ML 모델 배포 경험

- 주택자금 대출 지원
- 자녀 보육비 지원

- 크로스 브라우저 호환성 확보
- 서비스 A/B 테스트 구현

### 자격 요건

- HTML, CSS, JavaScript(ES6+) 기본기 탄탄한 분
- React 사용 경험 (개인 프로젝트 포함)

- Git을 활용한 협업 경험

### 우대 사항

- Next.js 사용 경험
- 상태 관리 라이브러리 경험 (Recoil, Zustand, Redux 등)
- 웹 접근성(WCAG) 이해 및 적용 경험
- 오픈소스 기여 경험
- 사이드 프로젝트 또는 해커톤 참여 경험

### 복리후생

- 어학 학원비 지원 (월 20만원)

- Tableau, Looker 등 BI 도구 사용 경험
- 데이터 분석 경력 2년 이상

### 우대 사항

- 글로벌 서비스 분석 경험
- 영어 또는 일어 커뮤니케이션 가능

## 공고 5: 쿠팡 - DevOps 엔지니어 (경력 3년 이상)

**회사명**: 쿠팡
**직무**: DevOps / 인프라 엔지니어
**고용형태**: 정규직
**근무지**: 서울특별시 송파구 (잠실)
**등록일**: 2025-02-05
**마감일**: 상시 채용

### 직무 소개

쿠팡 인프라팀은 국내 최대 이커머스 플랫폼의 안정적인 운영을 책임집니다. 수십만 개의 서버와 컨테이너를 관리하고, CI/CD 파이프라인을 통해 하루에도 수백 번의 배포가 안전하게 이루어지도록 시스템을 구축합니다.

### 주요 업무

- Kubernetes 기반 컨테이너 오케스트레이션 운영
- Terraform을 활용한 Infrastructure as Code 관리
- CI/CD 파이프라인 구축 (Jenkins, GitHub Actions)
- 모니터링 시스템 구축 (Prometheus, Grafana, ELK)
- 보안 취약점 점검 및 조치

### 자격 요건

- Linux 시스템 관리 경험 3년 이상
- AWS, GCP 등 클라우드 서비스 실무 경험
- Docker, Kubernetes 운영 경험
- 쉘 스크립트 또는 Python 자동화 스크립트 작성 능력
- 네트워크 기본 이해 (TCP/IP, DNS, Load Balancer)

### 우대 사항

- CKA, AWS SAA 등 관련 자격증 보유
- 대규모 트래픽 환경에서의 인프라 운영 경험
- 보안(ISO 27001, SOC2) 인증 관련 업무 경험
- GitOps 방법론 이해 및 적용 경험

### 복리후생

- 영어 공용어 환경 (글로벌 팀 협업)
- 무제한 간식 및 식사 제공
- 건강보험 및 의료비 지원
- 재택근무 가능
- 스톡 그랜트 제공
- 자기계발 지원 (컨퍼런스, 도서 등)

### 전형 절차

서류 전형 → 기술 면접 (시스템 설계) → 실무 면접 → HR 면접 → 최종 합격
