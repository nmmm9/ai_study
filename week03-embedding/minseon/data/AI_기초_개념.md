# AI 기초 개념 정리

## 인공지능(AI)이란?

인공지능은 인간의 지능적인 행동을 컴퓨터로 모방하는 기술입니다. 학습, 추론, 문제 해결, 언어 이해, 지각 등의 능력을 포함합니다.

## 머신러닝(Machine Learning)

머신러닝은 AI의 한 분야로, 데이터로부터 패턴을 학습해 예측하거나 결정을 내리는 알고리즘을 연구합니다.

- **지도학습**: 정답이 있는 데이터로 학습 (이미지 분류, 스팸 필터 등)
- **비지도학습**: 정답 없이 데이터의 패턴 발견 (군집화, 차원 축소 등)
- **강화학습**: 환경과 상호작용하며 보상을 최대화하는 방향으로 학습

## 딥러닝(Deep Learning)

딥러닝은 여러 층(layer)의 신경망을 사용하는 머신러닝입니다. 이미지 인식, 자연어 처리, 음성 인식 등에서 뛰어난 성능을 보입니다.

- **CNN(합성곱 신경망)**: 이미지 처리에 특화
- **RNN(순환 신경망)**: 순서가 있는 데이터(시계열, 텍스트) 처리
- **Transformer**: 현재 NLP의 핵심 구조, 어텐션 메커니즘 기반

## 자연어 처리(NLP)

자연어 처리는 컴퓨터가 인간의 언어를 이해하고 생성하는 기술입니다.

주요 태스크:
- 텍스트 분류
- 개체명 인식(NER)
- 기계 번역
- 질의응답
- 요약

## 대규모 언어 모델(LLM)

LLM은 수십억 개의 파라미터를 가진 대형 언어 모델입니다. GPT, Claude, Gemini 등이 대표적입니다.

특징:
- **사전학습(Pre-training)**: 대규모 텍스트 데이터로 언어 패턴 학습
- **파인튜닝(Fine-tuning)**: 특정 태스크에 맞게 추가 학습
- **인컨텍스트 러닝**: 프롬프트 내 예시만으로 새로운 태스크 수행

## 임베딩(Embedding)

임베딩은 텍스트(단어, 문장, 문서)를 숫자 벡터로 변환하는 기술입니다. 의미가 비슷한 텍스트는 벡터 공간에서 가까이 위치합니다.

- **word2vec**: 단어 임베딩의 시초
- **BERT**: 문맥을 고려한 문장 임베딩
- **text-embedding-3-small**: OpenAI의 최신 임베딩 모델 (1536차원)

## RAG(Retrieval-Augmented Generation)

RAG는 외부 문서를 검색해 LLM의 답변 품질을 높이는 기법입니다.

과정:
1. 문서를 청크로 분할
2. 각 청크를 임베딩 벡터로 변환
3. 질문도 임베딩 변환
4. 코사인 유사도로 관련 청크 검색
5. 검색된 청크를 컨텍스트로 LLM에 전달
6. LLM이 컨텍스트 기반으로 답변 생성
