## 공고 1: 카카오 — 백엔드 개발자 (서버플랫폼)

### 회사 소개
카카오는 대한민국 대표 IT 기업으로 카카오톡, 카카오맵, 카카오페이 등 다양한 플랫폼 서비스를 운영합니다. 판교 본사를 중심으로 국내외 수천만 사용자에게 서비스를 제공하며, 기술 중심의 수평적 조직 문화를 추구합니다.

### 직무 소개
카카오 서버플랫폼팀에서 대규모 트래픽을 처리하는 백엔드 시스템을 설계하고 개발합니다. 카카오톡 메시지 처리, 알림 시스템, API 게이트웨이 등의 핵심 인프라를 담당합니다.

### 주요 업무
- 카카오 공통 서버 플랫폼 설계 및 개발
- 대용량 트래픽 처리를 위한 분산 시스템 구현
- REST API 및 gRPC 기반 서비스 개발
- 성능 모니터링 및 장애 대응 시스템 구축
- 코드 리뷰 및 기술 문서 작성

### 자격 요건
- Java 또는 Kotlin 기반 백엔드 개발 경험 3년 이상
- Spring Boot, Spring Cloud 프레임워크 사용 경험
- MySQL, Redis 등 데이터베이스 운영 경험
- Git 기반 협업 경험
- 컴퓨터공학 또는 유관 전공 학사 이상

### 우대 사항
- 대규모 분산 시스템 설계 경험
- Kafka, RabbitMQ 등 메시지 큐 사용 경험
- Docker, Kubernetes 운영 경험
- 오픈소스 기여 경험

### 복리후생
- 유연근무제 (코어타임 10:00~16:00)
- 재택근무 가능 (주 2회)
- 식대 월 20만원 지원
- 자기계발비 연 200만원
- 건강검진 지원
- 사내 카페 운영

### 전형 절차
서류 전형 → 코딩 테스트 → 1차 기술 면접 → 2차 임원 면접 → 최종 합격

## 공고 2: 네이버 — AI/ML 엔지니어 (클로바팀)

### 주요 업무
- LLM 모델 서빙 인프라 설계 및 구현
- MLOps 파이프라인 자동화 (학습/평가/배포)
- 모델 성능 최적화 (양자화, 증류, 추론 속도 개선)
- A/B 테스트 프레임워크 운영
- AI 모델 품질 모니터링 시스템 구축

### 자격 요건
- Python 기반 ML/DL 개발 경험 3년 이상
- PyTorch 또는 TensorFlow 프레임워크 활용 경험
- Transformer 기반 모델 이해 및 활용 경험
- 분산 학습 환경 구성 경험 (DDP, FSDP 등)
- Linux 환경 개발 경험

### 우대 사항
- LLM Fine-tuning 경험 (LoRA, QLoRA 등)
- Kubernetes 기반 ML 클러스터 운영 경험
- CUDA 프로그래밍 경험
- 논문 구현 및 연구 경험

### 복리후생
- 성과급 및 스톡옵션 제공
- 연구비 지원 (학회 참가, 논문 구독)
- 사내 어린이집 운영

## 공고 3: 토스 (Viva Republica) — 프론트엔드 개발자

### 회사 소개
토스는 2015년 설립된 핀테크 스타트업으로, 간편송금 서비스로 시작해 현재 토스증권, 토스보험, 토스뱅크 등 종합 금융 플랫폼으로 성장했습니다. 월간 활성 사용자 2,000만 명 이상의 국내 대표 핀테크 기업입니다.

### 직무 소개

### 주요 업무
- 토스 웹/앱 프론트엔드 개발 및 유지보수
- 디자인 시스템 구축 및 컴포넌트 라이브러리 개발
- 성능 최적화 및 접근성 개선
- 금융 데이터 시각화 컴포넌트 개발
- A/B 테스트 및 사용자 행동 분석

### 자격 요건
- React 기반 프론트엔드 개발 경험 3년 이상
- TypeScript 사용 경험
- HTML/CSS 심층 이해
- RESTful API 연동 경험
- Git 협업 경험

### 우대 사항
- React Native 개발 경험
- 성능 최적화 경험 (Core Web Vitals)
- 테스트 코드 작성 경험 (Jest, Testing Library)

## 공고 4: 라인 (LINE Plus) — 데이터 엔지니어

### 회사 소개
라인은 일본, 대만, 태국 등 아시아 전역에서 메신저 서비스를 운영하는 글로벌 IT 기업입니다. 국내 성남 판교에 LINE Plus 법인을 두고 있으며, 글로벌 스케일의 데이터 플랫폼을 운영합니다.

### 직무 소개
LINE의 글로벌 데이터 플랫폼팀에서 수억 명의 사용자 행동 데이터를 처리하는 파이프라인을 설계하고 운영합니다. 실시간 스트리밍과 배치 처리를 모두 다루는 엔지니어링 포지션입니다.

### 주요 업무
- 대용량 데이터 파이프라인 설계 및 개발 (Kafka, Spark, Flink)
- 데이터 레이크하우스 아키텍처 구축 및 운영
- ETL/ELT 프로세스 자동화
- 데이터 품질 모니터링 및 이상 탐지 시스템 개발
- 비즈니스 팀과의 데이터 요구사항 분석 및 구현

### 자격 요건
- Python 또는 Scala 기반 데이터 엔지니어링 경험 3년 이상
- Apache Spark 또는 Flink 활용 경험
- SQL 고급 쿼리 작성 능력
- 분산 데이터베이스 이해 (HBase, Cassandra 등)
- 영어 커뮤니케이션 기본 가능

### 우대 사항
- 글로벌 서비스 데이터 처리 경험 (일 수십억 건 이상)
- Airflow, dbt 등 데이터 오케스트레이션 경험
- Delta Lake, Apache Iceberg 경험
- 일본어 또는 중국어 가능자

### 복리후생
- 글로벌 수준 연봉 및 인센티브
- 해외 법인 교류 프로그램
- 일어/영어 교육 지원
- 통근 버스 (판교/강남)
- 의료비 지원
- 자녀 학자금 지원

### 전형 절차
서류 전형 → 온라인 코딩 테스트 → 기술 면접 (1~2회) → HR 면접 → 최종 합격
지원 마감: 2024년 12월 15일

## 공고 5: 쿠팡 — DevOps/SRE 엔지니어

### 회사 소개
쿠팡은 국내 최대 이커머스 기업으로 로켓배송, 로켓프레시, 쿠팡플레이 등 다양한 서비스를 운영합니다. 연간 수조 원의 거래가 이루어지는 플랫폼의 안정성을 책임지는 SRE 팀을 모집합니다.

### 직무 소개
쿠팡 인프라 신뢰성 팀에서 전국 물류 센터와 연동되는 대규모 시스템의 안정성을 확보합니다. 장애 대응, 용량 계획, 배포 자동화 등 SRE 전반의 업무를 담당합니다.

### 주요 업무
- 서비스 가용성 및 성능 SLI/SLO 정의 및 관리
- CI/CD 파이프라인 구축 및 최적화
- 인프라 IaC 구현 (Terraform, Ansible)
- 장애 탐지 및 자동 복구 시스템 개발
- 용량 계획 및 비용 최적화

### 자격 요건
- Linux 시스템 운영 경험 3년 이상
- AWS 또는 GCP 클라우드 운영 경험
- Python 또는 Go 스크립팅 능력
- Docker, Kubernetes 운영 경험
- 모니터링 도구 운영 경험 (Prometheus, Grafana)

### 우대 사항
- 대규모 Kubernetes 클러스터 운영 경험 (1,000노드 이상)
- 장애 대응 사후 분석(Post-mortem) 작성 경험
- Istio, Envoy 등 서비스 메시 경험
- 영어로 기술 문서 작성 가능

### 복리후생
- 글로벌 기준 경쟁력 있는 연봉
- RSU (제한 조건부 주식) 지급
- 원격 근무 가능
- 서울 강남 사무실 (신규)
- 쿠팡이츠 배달비 지원
- 건강검진 및 심리상담 지원

### 전형 절차
이력서 검토 → 전화 스크리닝 → 기술 면접 (시스템 설계) → 온사이트 면접 → 오퍼
지원 마감: 상시 채용

- GMP 규정에 따른 시험 기록 및 데이터 검토
- 일탈 조사 (Out of Specification) 및 원인 분석
- 시험 방법 밸리데이션 수행 및 보고서 작성
- 글로벌 규제 기관(FDA, EMA) 실사 대응

### 우대 사항

### 복리후생
