# 한국장학재단 국가장학금

## 개요
대학생의 등록금 부담을 줄이기 위해 소득분위에 따라 차등 지급하는 정부 장학금이다. 1유형(소득연계)과 2유형(대학연계)으로 구분된다.

## 1유형 (소득연계)

### 지원 대상
- 대한민국 국적의 국내 대학 재학생 (신입, 편입, 재입학 포함)
- 소득분위 8분위 이하
- 나이: 만 39세 이하 (입학 당시 기준)

### 성적 기준
- 신입생: 성적 기준 없음 (첫 학기)
- 재학생: 직전 학기 12학점 이상 이수, 평점 B학점(80점) 이상
- 기초/차상위: 직전 학기 12학점 이상 이수, 평점 C학점(70점) 이상

### 지원 금액 (학기당)
| 소득분위 | 지원 금액 |
|----------|----------|
| 기초/차상위 | 전액 (등록금 전부) |
| 1분위 | 전액 |
| 2분위 | 전액 |
| 3분위 | 전액 |
| 4분위 | 등록금의 50% |
| 5분위 | 등록금의 50% |
| 6분위 | 등록금의 50% |
| 7분위 | 175만원 |
| 8분위 | 175만원 |

### 신청 방법
1. 한국장학재단 홈페이지 (www.kosaf.go.kr) 접속
2. 회원가입 후 국가장학금 신청
3. 가구원 동의 및 서류 제출
4. 소득분위 산정 (약 4~6주 소요)
5. 심사 후 장학금 지급

### 신청 기간
- 1학기: 전년도 11월 ~ 당해 3월 (1차/2차 나눠서 접수)
- 2학기: 당해 5월 ~ 9월

## 2유형 (대학연계)

### 지원 대상
- 소득분위 9분위 이하
- 각 대학이 자체 기준으로 선발

### 특징
- 대학이 등록금 인상을 자제하는 조건으로 정부 재원을 받아 운영
- 대학별로 선발 기준과 금액이 다름
- 별도 신청 없이 국가장학금 1유형 신청 시 자동 심사

## 주의사항
- 국가장학금은 등록금을 초과하여 지급되지 않음
- 교내/외 장학금과 합산하여 등록금 범위 내에서 지급
- 대학원생은 신청 불가
- 해당 학기 휴학 시 지급 불가
- 재학 중 장학금 수혜 횟수 제한 없음 (매 학기 신청 필요)
