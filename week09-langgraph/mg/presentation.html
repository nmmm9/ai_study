LangGraph 핵심 4개념

① State

그래프 전체에 흐르는 공유 데이터 (TypedDict).
Annotated[list, add] 같은 리듀서가 머지 방식을 결정.

② Node

State를 받아 부분 업데이트만 반환하는 함수.
전체 State 반환하면 reducer가 중복 머지.

③ Edge

일반 add_edge / 조건부 add_conditional_edges.
list 반환 시 여러 노드 동시 실행 (Send API).

④ Checkpointer

thread_id 단위 자동 영속화.
InMemorySaver → SqliteSaver 한 줄로 교체.

State 정의 — 리듀서가 가장 중요

from typing import Annotated, TypedDict
from operator import add

class GraphState(TypedDict):
    question: str                          # 덮어쓰기
    plan: list[str]                       # 덮어쓰기
    tool_results: Annotated[list[dict], add]   # ★ 누적 (concat)
    completed_agents: Annotated[list[str], add] # ★ 누적
    final_answer: str

함정: 노드는 수정한 키만 반환할 것. Annotated[list, add] 가 붙은 키에 전체 리스트를 반환하면 리듀서가 두 번 더해서 중복됨.

여러 도메인 에이전트가 병렬로 tool_results에 append → reducer가 자동 머지.

Supervisor — LLM에게 라우팅 위임

async def supervisor_node(state: dict) -> dict:
    response = await client.chat.completions.create(
        model="gpt-5-nano",
        messages=[
            {"role": "system", "content": ROUTING_PROMPT},
            *state["history"],            # ← 이전 대화 주입
            {"role": "user", "content": state["question"]},
        ],
        response_format={"type": "json_object"},
    )
    parsed = json.loads(response.choices[0].message.content)
    return {"plan": parsed["agents"]}     # 부분 업데이트만 반환

JSON Mode로 LLM이 {"agents": ["lifestyle", "sports"]} 만 뱉도록 강제. history를 함께 주면 "거기", "아까" 같은 후속 질문도 정확히 라우팅.

Conditional Edge — 병렬 분기

def route(state: GraphState) -> list[str]:
    plan = state.get("plan") or []
    if not plan:
        return ["writer"]
    return list(plan)              # ★ list 리턴 → 동시 실행

graph.add_conditional_edges(
    "supervisor",
    route,
    {"shopping": "shopping", "lifestyle": "lifestyle",
     "sports": "sports", ...},
)

# 모든 도메인 → Writer로 합류 (Fan-in)
for d in DOMAINS:
    graph.add_edge(d, "writer")

route()가 ["lifestyle", "sports"] 반환 → 두 노드 동시 실행. 각자 끝나면 reducer가 결과를 머지하고 Writer로 fan-in.

대화 메모리 — LangGraph Checkpointer

from langgraph.checkpoint.memory import InMemorySaver
from langgraph.graph import StateGraph, END

class ConvState(TypedDict):
    messages: Annotated[list[dict], add]   # 누적 reducer

# 1-노드 더미 그래프 — 순수 저장소로 활용
builder = StateGraph(ConvState)
builder.add_node("turn", lambda _: {})       # 빈 dict 리턴 (중복 방지)
graph = builder.compile(checkpointer=InMemorySaver())

# 저장
graph.invoke({"messages": [...]}, config={"configurable": {"thread_id": "t1"}})

# 복원
state = graph.get_state({"configurable": {"thread_id": "t1"}})

한 줄 교체로 영속화: InMemorySaver() → SqliteSaver(...) → PostgresSaver(...). 프론트엔드는 thread_id만 보내면 자동 복원.
